# RocketMQ

由于不支持 Java 9 及以上版本，暂时空缺

## Windows 下载安装 RocketMQ

- 开发环境：JDK

1. 下载

```
http://archive.apache.org/dist/rocketmq/
```

2. 解压

```
tar -xvzf rocketmq-all-4.7.1-bin-release.zip
```

3. PATH、ROCKETMQ_HOME 设置
4. 启动服务

```powershell
> mqnamesrv.cmd -n localhost:9876
```
